# typora 快捷键

refer to CheatSheet (macos)

| cmd + 1.....6      | heading 1......6      |
| :----------------- | --------------------- |
| cmd + =            | 提升标题级别          |
| cmd + -            | 降低标题级别          |
| Cmd + t            | 新标签页              |
| opt cmd + -        | 水平分割线            |
| Cmd + /            | 源码模式              |
| ctrl + `           | `code`                |
| opt cmd +c         | code block            |
| opt cmd +x         | task list             |
| Ctrl + cmd + 1,2,3 | 大纲, 文档列表,文件树 |
| opt + cmd + t      | table                 |
| Ctrl + enter       | 表格追加一行          |
